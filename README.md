# Google API V3 for Salesforce

This project is an api wrapper for Google API V3.  
Currently it contains only Calendar API.

## Calendar API

https://developers.google.com/google-apps/calendar/v3/reference/

### Implemented

- Acl
    - delete
    - get
    - insert
    - list
    - patch
    - update
- CalendarList
    - list
    - get
- Calendars
    - delete
    - get
    - insert
    - update
    - patch
    - clear
- Events
    - delete
    - get
    - list
    - insert
    - patch
    - update
    - quickAdd

### Not implemented

- CalendarList
    - delete
    - insert
    - patch
    - update
- Colors
- Events
    - import
    - instances
    - move
    - watch
- Freebusy
- Settings
- Channels
